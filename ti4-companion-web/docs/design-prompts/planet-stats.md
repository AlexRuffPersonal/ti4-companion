# Planet Stat Icons

Destination: `ti4-companion-web/public/icons/planet/`

These icons keep their game colours — resource (gold gear) and influence (blue wave) are colour-coded on every physical TI4 tile and players identify them instantly by colour. They appear inline next to numbers on planet cards so must be legible at 20x20px. Transparent background on all.

---

### resource.svg

Flat minimalist SVG vector icon for the Resource value symbol in Twilight Imperium 4. In the physical TI4 game the Resource symbol is a yellow gear or cog. Reproduce this as a clean geometric gear: a circle with 8 to 10 evenly spaced rectangular teeth protruding outward and a hollow circle cut from the centre. This is the industry and production value of a planet. Fill: #d4a017 (gold). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.

---

### influence.svg

Flat minimalist SVG vector icon for the Influence value symbol in Twilight Imperium 4. In the physical TI4 game the Influence symbol is a blue stylised broadcast wave — concentric arcs suggesting a signal or political reach. Reproduce this as three concentric arcs opening upward (like a wifi or broadcast symbol rotated so arcs open to the top) sitting above a small diamond or dot anchor point at the base. This is the political and vote value of a planet. Fill: #58a6ff (blue). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.
