# Command Token Icons

Destination: `ti4-companion-web/public/icons/tokens/`

Command tokens are faction-coloured in the physical game, but in the app these icons represent the three token pools (Tactic, Fleet, Strategy) abstractly — they need to be distinguishable by shape alone. The outer ring is white so it can be CSS-tinted to faction colour; the inner motif uses the pool's associated accent colour to remain identifiable regardless of faction tint. Transparent background on all.

---

### tactic.svg

Flat minimalist SVG vector icon of a Twilight Imperium 4 Tactic Command Token. A circular token shape — a clean thick outer ring border suggesting a physical coin or token — with a bold angular motif inside suggesting speed and tactical movement: a forward-pointing chevron or stylised directional arrow made of sharp angular lines, like a military rank chevron or a fast strike vector. Outer ring: white (#ffffff). Inner motif: #d4a017 (gold). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.

---

### fleet.svg

A circular token shape — a clean thick outer ring border suggesting a physical coin or token — with a motif inside: three small ship silhouettes (simple arrowhead or wedge shapes) arranged in a V-formation from above, like a fleet in formation. Outer ring: white (#ffffff). Inner motif: #58a6ff (blue). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.

### strategy.svg

Flat minimalist SVG vector icon of a Twilight Imperium 4 Strategy Command Token. A circular token shape — a clean thick outer ring border suggesting a physical coin or token — with a motif inside: an eight-pointed compass rose or star burst with alternating long and short points, centred within the circle, suggesting strategic planning and initiative.
