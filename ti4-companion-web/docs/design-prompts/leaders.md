# Leader Type Icons

Destination: `ti4-companion-web/public/icons/leaders/`

Leader type icons keep their colours — these appear as small type-badges on leader cards and use colour to reinforce the category meaning (Agent = blue operative, Commander = gold authority, Hero = red legendary power). Transparent background on all.

---

### agent.svg

Flat minimalist SVG vector icon for the Agent leader type in Twilight Imperium 4. An Agent is a diplomat or specialist who can be exhausted to use their ability on any player's turn — always watching, always ready to act. Icon concept: a stylised eye inside a circle — a bold open eye shape centred within a circular border, suggesting observation, intelligence, and readiness. Fill: #58a6ff (blue). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.

---

### commander.svg

Flat minimalist SVG vector icon for the Commander leader type in Twilight Imperium 4. A Commander is unlocked by meeting a faction condition and then provides a passive ongoing ability — permanent authority once earned. Icon concept: three bold chevrons stacked in a V pointing upward, like military sergeant's stripes or a general's rank insignia, suggesting permanent command authority. Fill: #d4a017 (gold). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.

---

### hero.svg

Flat minimalist SVG vector icon for the Hero leader type in Twilight Imperium 4. A Hero is unlocked by scoring 3 or more objectives and is purged permanently after a single use — a legendary one-time super-ability. Icon concept: a bold starburst or radiant crown — a pentagon or diamond shape at the centre with sharp rays emanating outward in all directions, suggesting legendary power used once and gone. Fill: #f85149 (red). Transparent background. 64x64px canvas. Flat vector, no gradients, no text.
